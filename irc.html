LiberaChat Basics

Connecting without installing anything.
If you cannot or do not wish to install a client on your computer, you can use a web client.
Just select the top right conner "Connect" button and sellect WebChat.

They also provide two web clients:
KiwiIRC, a robust web client
gamja, a lightweight web client

OR you can use IRCCloud is a popular web client service that keeps you connected
after closing the page and has a limited free plan. It also has iOS and Android apps.

Libera.Chat is now hosting the IRC channel for #jaxlug

You need to install a IRC chat client program like HexChat in your machine.
Setup a user name and nickname. Asper your prefered client program.
login into your client program and search for the #jaxlug channel.

  • Depending on your IRC client program you might have to register your "nickname".
    some will require a confirmation email to be sent.

  • Copy the line in the email that looks like this:
    /msg NickServ VERIFY REGISTER into the chat window, and you should see something like:
    has now been verified.
    That's it!
