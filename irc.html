Connecting to JaxLUG IRC without installing a client.

If you cannot OR do not wish to install a IRC client in your computer, you can use a Libera.Chat web client.
Just visit Libera.Chat and select the top right corner "Connect" button and sellect WebChat.

Libera.Chat can also use two web clients:

    KiwiIRC.com, a robust Web client. https://kiwiirc.com/
    Emersion, gamja. a lightweight web client https://sr.ht/~emersion/gamja/

OR you can use IRCCloud is a popular web client service that keeps you connected
after closing the page and has a limited free plan. It also has iOS and Android apps.

Libera.Chat Settings

You need to install a IRC chat client program like HexChat, KVIRC, or Polari in your Linux machine.
for Windows IRC clients look for a Windows Blog.
Setup a user name, nickname, and password. As per your referred IRC client program.
login into your IRC client program and search for the #jaxlug channel.

  • Depending on your IRC client program you might have to register your "nickname".
    some IRC client will required you to confirm your email and Nickname.

  • Copy the line in the email that looks like this:
    /msg NickServ VERIFY REGISTER "nickname"
    into the chat window, and you should see a message that looks like:
    "nickname" has now been verified.
    That's it!
