Basic Command Line Knowledge

Command The ls command (list directory content)
-options -l = long format
-h = human readable
-S = Sort by Size
-a = List all files
Command The cp command (list directory content)
-options cp item1 item2 -l = long format
-r = --recursive
-v = --verbose
